# 说明
1、本程序为爬取指定地区招标、采购、意向、变更、中标信息，并定时自动发送到钉钉群。已录入的网站见下表。
2、目前只推送招标、意向、意向变更信息。
3、招标变更感觉作用不大，实际未运行。
4、TODO: 中标信息目前只做爬取还没做推送，整理成表格单点推送或许更好点儿。

# 安装依赖
requirement.txt里面可能不全，运行报错就装什么依赖吧

# 配置文件
1、复制 config_bak.py 为 config.py，并补充其中的配置，主要是城市、钉钉机器人url等
2、每天定时发送时间需修改send.py文件中的send函数中内容，默认每天08:05

# 部署指南
1、因部分网站需有头浏览器，建议windows server环境并使用edge或谷歌浏览器部署，否则大唐商务、河北采购的中标信息无法爬取（未验证）。
2、正式运行前，先修改并运行set_edge.py文件以设置浏览器路径。
3、运行main.py。
